# Glossario

## Termos de negocio

- **Jornada base**: carga diaria esperada (8h regular, 4h estagiario).
- **Horas feitas**: horas dentro da jornada base.
- **Horas extras (HE)**: excedente da jornada base.
- **Dia finalizado**: estado final do registro diario.
- **Alteracao manual**: ajuste de horario feito apos registro.

## Termos operacionais

- **Item pai**: registro do colaborador dentro do grupo mensal.
- **Subitem**: lancamento diario de ponto.
- **Grupo mensal**: agrupamento de registros por mes e ano.
- **Corte de fechamento**: horario limite para encerramento automatico (18:00).

## Termos tecnicos

- **Chave tecnica de coluna**: identificador usado pela API para ler/escrever campos.
- **Unknown column**: erro de coluna inexistente ou mapeamento incorreto.
- **Timezone shift**: deslocamento de data por parse UTC.
- **Board SDK**: camada de acesso aos boards Monday.com.
- **GraphQL mutation**: operacao de escrita usada para criar grupo.
