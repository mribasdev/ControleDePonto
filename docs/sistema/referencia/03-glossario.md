# Glossário

## Termos de negócio

- **Jornada base**: carga diária esperada (8h regular, 4h estagiário).
- **Horas feitas**: horas dentro da jornada base.
- **Horas extras (HE)**: excedente da jornada base.
- **Dia finalizado**: estado final do registro diário.
- **Alteração manual**: ajuste de horário feito após registro.

## Termos operacionais

- **Item pai**: registro do colaborador dentro do grupo mensal.
- **Subitem**: lançamento diário de ponto.
- **Grupo mensal**: agrupamento de registros por mês e ano.
- **Corte de fechamento**: horário limite para encerramento automático (18:00).

## Termos técnicos

- **Chave técnica de coluna**: identificador usado pela API para ler/escrever campos.
- **Unknown column**: erro de coluna inexistente ou mapeamento incorreto.
- **Timezone shift**: deslocamento de data por parse UTC.
- **Board SDK**: camada de acesso aos boards Monday.com.
- **GraphQL mutation**: operação de escrita usada para criar grupo.
